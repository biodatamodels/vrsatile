Class: VariationDescriptor

This descriptor class is used for describing VRS Variation value objects.

URI: GA4GHValueObjectDescriptorDefinitions:VariationDescriptor

Slots

Name Range Cardinality Description Info
id CURIE 1..1 Descriptor ID; MUST be unique within document. .
vrs_type TypeOptions 1..1 MUST be "VariationDescriptor". .
label NONE 0..1 A primary label for the value object. .
description NONE 0..1 A free-text description of the value object. .
xrefs CURIE 0..* None .
alternate_labels NONE 0..* None .
extensions Extension 0..* None .
variation_id CURIE 0..1 This SHOULD be provided if variation is omitted. .
variation Variation 1..1 This SHOULD be provided if variation_id is omitted. .
molecule_context MoleculeContextOptions 0..1 The molecular context of this variant. Must be one of "genomic", "transcript", or "protein". .
structural_type CURIE 0..1 The structural variant type associated with this variant. We RECOMMEND a descendent term of SO:0001537. .
expressions Expression 1..* Typically HGVS or ISCN nomenclature expressions. Other systems relevant to the description of variation MAY be used. .
vcf_record VcfRecord 0..1 A VCF Record of the variant. This SHOULD be a single allele, the VCF genotype (GT) field should be represented in the allelic_state attribute. .
gene_context NONE 0..1 A specific gene context that applies to this variant. .
vrs_ref_allele_seq Sequence 0..1 A VRS Sequence corresponding to a "ref allele", describing the sequence expected at a VRS SequenceLocation reference. .
allelic_state CURIE 0..1 We RECOMMEND that the allelic_state of a variation be described by terms from the Genotype Ontology (GENO). These SHOULD descend from concept GENO:0000875 <http://purl.obolibrary.org/obo/GENO_0000875>. .
