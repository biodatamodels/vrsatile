Class: CategoricalVariationDescriptor

This descriptor class is used for describing Categorical Variation vrs_value objects.

URI: GA4GHValueObjectDescriptorDefinitions:CategoricalVariationDescriptor

Slots

Name Range Cardinality Description Info
id CURIE 1..1 Descriptor ID; MUST be unique within document. .
vrs_type NONE 1..1 MUST be "VariationDescriptor". .
label NONE 0..1 A primary label for the value object. .
description NONE 0..1 A free-text description of the value object. .
xrefs CURIE 0..* None .
alternate_labels NONE 0..* None .
extensions Extension 0..* None .
version NONE 0..1 The version of the Categorical Variation Descriptor. .
categorical_variation_id CURIE 0..1 This SHOULD be provided if categorical_variation is omitted. .
categorical_variation CategoricalVariation 0..1 This SHOULD be provided if variation_id is omitted. .
vrs_members VariationMember 0..* None .
